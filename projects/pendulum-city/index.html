Overview

Pendulum City simulates a field of pendulums that move at different phases to create emergent wave patterns. This page documents goals, approach, and outcomes.

This is a template you can adapt to any project. Replace copy and add screenshots later.

Problem

  • Animate hundreds of elements smoothly on modest hardware.
  • Keep the UI minimal and accessible.
  • Ship a clean red/black presentation.

Approach

CSS transforms + requestAnimationFrame, no heavy frameworks. Small, readable code and semantic HTML with CSS custom properties for theming.
